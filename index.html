Fire & Smoke Detection

A YOLOv8-based pipeline for training and running real-time fire and smoke detection models — built for edge deployment.

YOLOv8 PyTorch Python 3.9+ MIT License
View on GitHub Quickstart

Features

🧊

Two-phase transfer learning

A frozen-backbone warmup phase followed by a full fine-tune, with resume-from-checkpoint support.

⚙️

Hardware-aware training

Device, batch size, dataloader workers, and mixed precision are auto-detected from the installed GPU(s).

🔍

High small-object recall

1280px input, aggressive augmentation, and a low confidence threshold tuned to catch faint or distant smoke.

🛰️

Production inference

Supervised background workers, a bounded upload queue, worker restart, and graceful drain-on-shutdown.

🎛️

Fully env-configurable

Inference behavior is overridable via AGRO_* environment variables — same binary in dev and on-device.

📦

Two classes

Detects fire and smoke in standard YOLO-format datasets.

Quickstart

# 1. Install dependencies
pip install -r requirements.txt

# 2. Place your dataset under datasets/fire-8/ (YOLO format)
#    train/ valid/ test/ with images + labels

# 3. Train (auto-tunes to your GPU, two-phase)
python train_fire.py

# 4. Run real-time inference
python detect.py

Inference configuration

Override behavior at runtime with environment variables:

VariableDefaultDescription
AGRO_CAMERA0Camera index
AGRO_FIRE_CONF0.35Display confidence threshold
AGRO_CONF_UPLOAD0.80Confidence required to upload an event
AGRO_FIRE_IMGSZ1280Inference image size
AGRO_GPS_LAT / LNGunsetGPS coordinates attached to uploads
AGRO_LOG_LEVELINFOLogging verbosity
